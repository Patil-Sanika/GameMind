Steps : How to Play Game Mind

Step 1. Objective : Repeat the sequence of colors that Game is plays. The sequence starts with one color and adds one more each time you successfully repeat it.

Step 2. Game Start : Press start button to start the game.

Step 3. Game's Turn : Game will play a sequence of colors by lighting up the buttons in a specific order. Watch and remember the sequence of colors.

Step 4. Player's Turn : After game's turn, it's your turn to repeat the sequence. Click or tap the buttons in the same order that game is played them.

Step 5. Feedback : If you correctly repeat the sequence, game will add one more color to the sequence and play it again. If you make a mistake, the game ends.

Step 6. Winning : Continue repeating the sequence correctly to see how far you can go. The game gets harder as the sequence gets longer.

Step 8. Restart : To play again,Press start button to play a new round.
